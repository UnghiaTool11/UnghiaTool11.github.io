☁️ Hướng Dẫn Chạy Unghia_Enc
Trên CloudShell

1. Giới thiệu

Chào mừng bạn đến với hướng dẫn chi tiết về cách thiết lập và chạy tool UnghiaEnc trên Google Cloud Shell. Đây là một nền tảng miễn phí, mạnh mẽ và lý tưởng để bạn chạy các tác vụ từ xa mà không cần cài đặt phức tạp. Hướng dẫn này phù hợp cho người dùng trên mọi thiết bị, bao gồm PC, Android và iOS.

2. Truy cập CloudShell

  • Mở Google Cloud Shell trên trình duyệt của bạn bằng cách truy cập https://shell.cloud.google.com. Hoặc bạn có thể tải và mở từ ứng dụng Google Cloud trên điện thoại:
    • **Android:** Google Play Store
    • **iOS:** Apple App Store

3. Tải Unghia_Enc

3.1 curl -o Unghia_Enc.py https://raw.githubusercontent.com/UnghiaTool11/UnghiaTool11.github.io/refs/heads/main/Unghia_Enc.py

4. Khởi Chạy Tool

Khởi động công cụ và bắt đầu sử dụng.

4.1 python Unghia_Enc.py
Khởi chạy công cụ. Sau đó, Nhập tên file muốn obf.
**HOÀN THÀNH!** Bạn đã sẵn sàng để sử dụng tool Unghia_Enc.
